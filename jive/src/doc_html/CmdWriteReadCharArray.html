Command WriteReadCharArray :
    Send a given string to the socket and waits for a given number of
    binary bytes. Those received bytes are returned to the caller.



WriteReadCharArray Definition
Input Argument Tango::DEVVAR_LONGSTRINGARRAY str[0] = String to send
lg[0] = Number of bytes to be read
Output Argument Tango::DEVVAR_CHARARRAY Read bytes
