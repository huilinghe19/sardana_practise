Socket Class Commands
Name
Input type
Output type
Level
Description
State DEV_VOID DEV_STATE OPERATOR This command gets the device state (stored in its device_state data member) and returns it to the caller.
Status DEV_VOID CONST_DEV_STRING OPERATOR This command gets the device status (stored in its device_status data member) and returns it to the caller.
Reconnect DEV_VOID DEV_VOID OPERATOR Attempts to re-establish the socket connection.
Write DEV_STRING DEV_VOID OPERATOR Command used to send a string to the socket.
WriteAndRead DEV_STRING DEV_STRING OPERATOR Write to the socket and waits for the answer.
The end of the answer is triggered by the new line character!
WriteReadUntil DEVVAR_STRINGARRAY DEV_STRING OPERATOR Write to the socket and waits for the answer.
The answer is triggered by select() which indicates that the the anser has arrived.
WriteCharArray DEVVAR_CHARARRAY DEV_VOID OPERATOR write on socket input Argument.
Read DEV_VOID DEV_STRING OPERATOR Command used to read a string from the socket.
Readln DEV_VOID DEV_STRING OPERATOR Reads from a socket until a "\n" has been received.
ReadUntil DEV_STRING DEV_STRING OPERATOR This command read data from the socket until the terminator has been reached.
ReadCharArray DEV_LONG DEVVAR_CHARARRAY OPERATOR read on socket and return output Argument.
WriteReadCharArray DEVVAR_LONGSTRINGARRAY DEVVAR_CHARARRAY OPERATOR Send a given string to the socket and waits for a given number of
binary bytes. Those received bytes are returned to the caller.
